# Attandence System

Record user attandence based on IC

## Setup

1. Git clone the repo
    ```
    git clone https://github.com/haizadtarik/attandence-system.git
    ```

2. Install Dependencies
    ```
    pip install -r requirements.txt
    ```

## Run

1. Bring up the server
    ```
    python server.py
    ```
